HTML이란 무엇인가?

Hypertext Markup Language (HTML) is the standard Markup language for creating web pages and web application. Web browsers receive HTML documents from a web server or from local storage and render them into multimedia web pages. HTML describes the structure or a web page semantically and originally included cues for the appearance of the documents.

HTML elements are the building blocks of HTML pages. With HTML constucuts, images and other objects, suc as interacitve forms, may be embeded into rendered page.
